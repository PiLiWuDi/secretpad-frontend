# 项目初始化&启动

## 环境准备

确保正确安装 [Node.js](https://nodejs.org/en/) 且版本为 14+ 即可。

```bash
$ node -v
v14.19.1
```

确保已安装[pnpm](https://pnpm.io/installation#using-npm)

```bash
$ pnpm -v
7.29.1
```

## 初始化项目

在项目根目录运行

```bash
$ pnpm bootstrap
```

## 启动项目

在项目根目录运行

```bash
$ pnpm dev
```

此时访问(http://localhost:8000) 即可进入平台
